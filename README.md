# 💬 NLP Sentiment Analysis System

## 📌 Project Overview

This project was developed as part of the Horizon TechX AI & Data Science Internship.

NLP Sentiment Analysis System is an Artificial Intelligence project that analyzes text data and classifies user opinions or reviews into Positive, Negative, and Neutral sentiments.

The system uses Natural Language Processing techniques to understand text emotions and provides meaningful insights from customer feedback through an interactive dashboard.

## 🚀 Features

- Text sentiment analysis
- Customer review analysis
- Positive, Negative, and Neutral classification
- Sentiment score calculation
- CSV dataset analysis
- Bulk review processing
- Sentiment visualization dashboard
- Interactive Streamlit interface

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- TextBlob
- Natural Language Processing (NLP)
- Data Visualization
- Sentiment Analysis Concepts

## 🧠 NLP Approach

Natural Language Processing allows computers to understand, analyze, and process human language.

In this project, NLP techniques are used to analyze customer reviews and determine the emotional meaning behind text data.

The sentiment analyzer evaluates text polarity and classifies the result into:

Positive Sentiment  
Neutral Sentiment  
Negative Sentiment


## 📈 Analysis Performed

Single Text Analysis

Analyzes individual user input and predicts sentiment with a sentiment score.


Dataset Sentiment Analysis

Processes multiple customer reviews from CSV files and generates sentiment-based insights.


Sentiment Visualization

Displays sentiment distribution to better understand customer opinions and feedback patterns.
